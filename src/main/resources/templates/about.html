Budget API CRUD Backend Service

This project is a Budget API CRUD service that provides essential functionality for managing budgets. This service is integrated with an Angular frontend and features dual authentication with Google OAuth and serves as a resource server for Okta.

Key Features:

  • CRUD Operations: Allows users to Create, Read, Update, and Delete budget entries.
  • Angular Frontend: Provides a user-friendly and responsive interface for managing budgets.
  • Dual Authentication: Offers dual authentication with Google OAuth for secure access.
  • Okta Integration: Serves as a resource server for Okta, enhancing security and access control.

Technologies Used:

  • Frontend: Angular
  • Backend: Java Spring
  • Authentication: Google OAuth and Okta

Project Status:

This project is currently in active development, with a focus on creating a robust budget management system that ensures data security, user-friendliness, and seamless integration with authentication providers.

Goals:

My goals for this project include delivering a fully functional and secure budget management system that simplifies budget tracking and enhances user experience.
